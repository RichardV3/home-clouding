# IRIS-VE v2.0 - Versione Migliorata

## 🎉 Nuove Funzionalità Implementate

### 🎨 Design e UX
- **Design Apple-inspired**: Interfaccia elegante e moderna in stile Apple/iOS
- **Theme Switch**: Commutatore tra tema chiaro e scuro con animazioni fluide
- **Responsive Design**: Perfettamente adattabile a tutti i dispositivi
- **Animazioni Smooth**: Transizioni e animazioni per un'esperienza premium

### 🔧 Funzionalità Core
- **Progress Bar**: Barre di caricamento per upload/download
- **Loader**: Indicatori di caricamento per ogni risorsa/pagina
- **Toast Notifications**: Notifiche per tutte le azioni (download/upload/eliminazione/creazione/modifica)
- **Filtri Cartelle**: Filtro tra cartelle normali e criptate
- **Conversione Dimensioni**: Conversione automatica delle dimensioni (1024KB → 1MB)

### 📝 Gestione Cartelle
- **Modifica Cartelle**: Possibilità di modificare nome e descrizione
- **Eliminazione Cartelle**: Funzione di eliminazione con conferma
- **Descrizioni**: Support per descrizioni nelle cartelle criptate

### 📄 Editor di Testi
- **Editor Integrato**: Editor testuale per file di testo
- **Sintassi Highlighting**: Supporto per diversi formati di file
- **Salvataggio Rapido**: Ctrl/Cmd+S per salvare rapidamente

### 📚 Documentazione
- **Visualizzazione**: Pulsante per visualizzare il file documentation.md
- **API Endpoint**: Nuovo endpoint /api/documentation per caricare il contenuto

### 🔐 Sicurezza e Accessibilità
- **Accessibilità**: Design accessibile con supporto per screen reader
- **Keyboard Shortcuts**: Scorciatoie da tastiera per azioni comuni
- **Validazione**: Validazione migliorata dei form

## 🚀 Installazione

1. Sostituisci i file esistenti con quelli in questo ZIP
2. Assicurati che tutte le dipendenze Python siano installate
3. Crea un file `documentation.md` nella root del progetto
4. Avvia l'applicazione con `python app.py`

## 📱 Compatibilità

- ✅ Desktop (Windows, macOS, Linux)
- ✅ Tablet (iPad, Android)
- ✅ Mobile (iPhone, Android)
- ✅ Tutti i browser moderni
